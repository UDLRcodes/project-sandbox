# Security Policy

## Supported versions

This project is developed on `main`. Security fixes are applied to `main`; please test
against the latest commit before reporting.

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Instead, report privately via GitHub's
[**Report a vulnerability**](https://github.com/UDLRcodes/project-sandbox/security/advisories/new)
(Security → Advisories → Report a vulnerability). Include steps to reproduce and the impact.

You can expect an acknowledgement within a few days. Thanks for helping keep the project safe.
